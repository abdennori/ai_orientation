# AI_Orientation 🎓
### المساعد الذكي للتوجيه الجامعي في الجزائر

مساعد محادثة (Chatbot) بالعربية، مبني بـ FastAPI، يساعد طلبة البكالوريا في الجزائر على:
- اقتراح تخصصات جامعية مناسبة حسب الشعبة والمعدل.
- مقارنة تخصصين في جدول واضح.
- شرح مفصّل لأي تخصص من **39 تخصصًا جزائريًا** (طبية، هندسية، علمية، اقتصادية، أدبية، وفنية).
- شرح نظام LMD، التسجيل الجامعي، المدارس العليا، الأقسام التحضيرية (CPGE)، التكوين المهني،
  الجامعات الخاصة، إعادة التوجيه، الدراسة والمنح بالخارج، والمنحة والإيواء.

## ⚙️ التثبيت والتشغيل

### 1) تثبيت المتطلبات

يُفضّل استخدام بيئة افتراضية (virtual environment):

```bash
python3 -m venv venv
source venv/bin/activate      # على Windows: venv\Scripts\activate

pip install -r requirements.txt
```

### 2) تشغيل الخادم

```bash
uvicorn app:app --reload
```

ثم افتح المتصفح على العنوان:

```
http://127.0.0.1:8000
```

## 🔑 إضافة مفتاح OpenAI API (اختياري)

بشكل افتراضي، يعمل التطبيق بمحرك محلي (Rule-Based) يعتمد بالكامل على `majors.json` دون الحاجة لأي اتصال بالإنترنت أو مفتاح API.

لتفعيل الردود المولّدة عبر OpenAI بدلاً من المحرك المحلي، **المفتاح موجود بالفعل جاهزًا في ملف `.env`** بجانب `app.py`:

```
OPENAI_API_KEY=sk-proj-....
```

`app.py` يقرأ هذا الملف تلقائيًا عند الإقلاع (عبر `python-dotenv`) ويفعّل `openai_generate_response()`
مع برومبت مخصص للتوجيه الجامعي، ويرجع تلقائيًا للمحرك المحلي في حال فشل الاتصال.

> ⚠️ **تنبيه أمني مهم:** المفتاح **لم يُكتب داخل الكود المصدري (`app.py`)** عمدًا، بل وُضع في ملف `.env`
> منفصل ومُدرج في `.gitignore` حتى لا يُرفع بالخطأ إلى GitHub أو أي مستودع عام. لا تشارك محتوى `.env` مع أحد،
> ولا ترفعه لأي مكان عمومي — أي شخص يحصل عليه يمكنه استعمال رصيدك على OpenAI. إذا شككت أن المفتاح تسرب،
> ألغه فورًا من [platform.openai.com/api-keys](https://platform.openai.com/api-keys) وأنشئ مفتاحًا جديدًا.
> عند النشر على استضافة مثل Render، استعمل قسم **Environment Variables** بدل رفع ملف `.env` (انظر أدناه).

> ملاحظة: تأكد من أن حزمتي `openai` و`python-dotenv` مثبّتتان (موجودتان أصلاً في `requirements.txt`).

## 🚀 نشر المشروع على Render

1. ادفع المشروع إلى مستودع GitHub.
2. أنشئ حسابًا على [render.com](https://render.com) واختر **New Web Service**.
3. اربط مستودع GitHub الخاص بالمشروع.
4. اضبط الإعدادات التالية:
   - **Build Command:** `pip install -r requirements.txt`
   - **Start Command:** `uvicorn app:app --host 0.0.0.0 --port $PORT`
   - **Environment:** Python 3.11 (أو أحدث)
5. إن أردت تفعيل OpenAI API، أضف متغير البيئة `OPENAI_API_KEY` من قسم **Environment Variables** في لوحة تحكم Render.
6. اضغط **Create Web Service**، وانتظر انتهاء عملية النشر — سيصبح تطبيقك متاحًا على رابط عام من Render.

## 🧠 كيف يعمل المحرك المحلي؟

الدالة الموحدة `generate_response()` في `app.py` هي نقطة الدخول:

```python
def generate_response(message: str) -> str:
    if OPENAI_API_KEY:
        return openai_generate_response(message)
    return local_generate_response(message)
```

`local_generate_response()` يحلل الرسالة عبر كلمات مفتاحية بسيطة (تدعم أيضًا بعض
صيغ الدارجة الجزائرية الشائعة) لتحديد النية من بين 14 نية ممكنة: تحية/شكر، مقارنة،
LMD، الأقسام التحضيرية، إعادة التوجيه، تسجيل، مدارس عليا، تكوين مهني، جامعات خاصة،
دراسة بالخارج، منحة وإيواء، فرص عمل عامة، اقتراح حسب الشعبة والمعدل، وشرح تخصص محدد
— ثم يبني الرد اعتمادًا على بيانات `majors.json` أو نصوص جاهزة لكل نية.

## 📌 ملاحظات

- الواجهة مبنية بالكامل بـ HTML/CSS/JavaScript الأصلي (بدون أي إطار عمل ضخم)، وتستخدم `fetch()` للتواصل مع الخادم دون إعادة تحميل الصفحة.
- التصميم يدعم اللغة العربية بالكامل مع اتجاه RTL، ووضعين ليلي وفاتح يُحفظ اختيارهما في `localStorage`.
- البيانات في `majors.json` هي بيانات إرشادية عامة (39 تخصصًا)، وقد تختلف بعض التفاصيل (المعدلات الدنيا، المدارس) من سنة لأخرى — يُنصح بالتحقق من المصادر الرسمية (وزارة التعليم العالي، الديوان الوطني للتعليم والتكوين عن بعد) قبل اتخاذ قرارات نهائية.
- **لا تنشر هذا المشروع (بما فيه ملف `.env`) في مستودع GitHub عمومي دون حذف/تجاهل `.env` أولاً** — راجع تنبيه المفتاح أعلاه.
